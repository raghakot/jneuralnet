Subversion

Use the following command to checkout the latest sources from subversion repository:

svn checkout https://jneuralnet.dev.java.net/svn/jneuralnet/trunk PRNAME --username username

(PRNAME can be JNeuralNet or JNNHelperNB)

 

Username: guest

Password: leave it blank.

(Developers can use their user id and password to check out)
