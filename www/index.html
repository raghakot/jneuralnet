Objectives

 

- To create a very easy to use yet powerful neural net creator and trainer

- Main emphasis will be on simplifying the training process as much as possible

  (including runtime suggestions on improving the training patterns)

 

 

About

 

- A framework for multi-layered supervised perception model in java

- A very simple to use framework currently meant only for supervised multilayered model

- Customization support for newer training algorithms
